Проверка на 400 Http Status code:

400: Неверный запрос. Это может быть вызвано различными действиями пользователя
401: Аутентификация не прошла.
403: Аутентифицированному пользователю не разрешен доступ к указанной конечной точке API.
404: Запрашиваемый ресурс не существует.
405: Метод не разрешен. Проверьте заголовок Allow на предмет разрешенных методов HTTP.
410: Запрашиваемый контент удалён с сервера.
412: Указаны условия в заголовках, которые сервер не может выполнить
415: Неподдерживаемый тип медиа. Запрошенный тип содержимого или номер версии недопустимы
422: Не удалось выполнить проверку данных (например, в ответ на POST-запрос).
429: Слишком много запросов. Запрос был отклонен из-за ограничения скорости.

Проверка на 500 Http Status code:

500: Внутренняя ошибка сервера. Это может быть вызвано внутренними ошибками программы. (Смоделировать и сломать то, что не покрыто в коде)
